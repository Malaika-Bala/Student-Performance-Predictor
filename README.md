Student Performance Predictor – A Machine Learning Console App (Python)

The Student Performance Predictor is a simple Python-based machine learning project that estimates a student’s final marks based on study habits and academic history. It demonstrates the practical use of basic AI/ML concepts in solving real-world educational problems.

Project Overview

Students often struggle to understand how factors like study time, attendance, and previous performance affect their final results.

This project provides a simple solution to:

Predict final marks using input data
Analyze the impact of study habits
Help students make better academic decisions

The project is beginner-friendly and aligns with the fundamentals of AI and ML.

Features
Predict Performance

Users input:

Study hours
Attendance
Previous marks

The model predicts final marks instantly.

Machine Learning Model
Uses Linear Regression
Trained on sample student dataset

Easy to run and use
No GUI required

How to Run the Program
1. Install Python (3.x)
2. Clone the repository
git clone https://github.com/YOUR_USERNAME/Student-Performance-Predictor.git
3. Navigate to folder
cd Student-Performance-Predictor
4. Install dependencies
pip install -r requirements.txt
5. Train the model
python model.py
6. Run the program
python main.py
